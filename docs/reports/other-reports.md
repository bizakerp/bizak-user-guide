# Other Reports

This category can hold supporting reports that do not fit neatly into the main operational buckets.

## Examples

- user activity log
- audit-style history
- operational summaries
- dynamic report outputs
- field operation summaries

## What these reports answer

- Who used the system and when?
- What happened in the audit trail?
- Which operational summary should the team review?

## Real report names in the app

| Report | Purpose |
| --- | --- |
| `user-activity` | User activity log |
| `user-activity-log` | Other user activity log view |
| `activity-log` | IRD activity log |

## Notes

This bucket is useful when a report does not belong to sales, procurement, inventory, or finance.
