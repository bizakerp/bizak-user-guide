# Other Reports

This category can hold supporting reports that do not fit neatly into the main operational buckets.

## Examples

- user activity log
- audit-style history
- operational summaries

## Real report names in the app

| Report | Purpose |
| --- | --- |
| `user-activity` | User activity log |
| `user-activity-log` | Other user activity log view |
| `activity-log` | IRD activity log |
