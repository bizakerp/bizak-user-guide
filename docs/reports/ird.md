# IRD Reports

This page covers IRD and tax reporting.
Use it for tax, register, and compliance views.

## Main routes in the app

| Route | Purpose |
| --- | --- |
| `materialized` | Materialized summary |
| `materialized-return` | Materialized return |
| `vat-report` | VAT register |
| `sales-book` | Sales book |
| `activity-log` | Activity log |
| `sales-utility` | Sales utility |
| `purchase-register` | Purchase register |
| `sales-register` | Sales register |
| `anusuchi-nine` | Anusuchi nine |

## Example

An accountant opens the VAT report or sales register when preparing tax work.
The IRD area keeps those tax views separate from the standard accounting reports.
