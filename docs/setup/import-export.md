# Import and Export

Bizak supports bulk import and data export to make setup and migration faster.

## Import examples

- master data
- opening balances
- inventory quantities
- receivables and payables
- sales and purchase transactions

## When import is useful

- initial go-live
- legacy migration
- bulk correction
- new branch setup
