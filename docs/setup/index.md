# Core Setup

This section covers the setup work needed before daily operations can run smoothly.

## Setup areas

- company details
- currencies and locations
- account and ledger setup
- tax setup
- users and organization structure
- preferences and numbering rules
- import and export

## Before you start

- Confirm the company structure is correct.
- Confirm the user has admin or setup access.
- Confirm the chart of accounts and locations are ready.
- Confirm naming rules for items, ledgers, and documents.

## Why setup matters

Most Bizak modules depend on setup data.
Items depend on units and categories.
Transactions depend on ledgers and roles.
Reports depend on a valid accounting structure.
