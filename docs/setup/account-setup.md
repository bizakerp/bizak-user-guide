# Account Setup

Account setup prepares the financial framework Bizak uses for posting and reporting.

## Key areas

- tax setup
- accounting periods
- account lists
- budget categories
- cost categories
- customer categories
- incoterms
- job types
- payment methods
- price levels
- terms
- vendor categories
- opening balances
- expense categories

## Why it matters

If account setup is incomplete, users may be blocked from posting transactions or generating accurate financial reports.
