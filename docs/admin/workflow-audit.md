# Workflow and Audit

Bizak tracks workflow-related activity so teams can review how a transaction moved through the system.

## Audit topics

- workflow history
- related records
- system notes
- activity logs

## What this helps with

- finding who created or approved a record
- checking which document led to another document
- reviewing comment history
- verifying the sequence of a transaction

## Example

A purchase order can move to receipt and then to vendor bill.
The workflow trail should make that chain visible.
