# Troubleshooting

This section will collect the most common Bizak user issues and the steps to resolve them.

## Typical topics

- login problems
- permission issues
- validation errors
- missing setup data
- posting failures
- report mismatches

## First checks

1. Confirm the user has the right role.
2. Confirm the required setup records exist.
3. Confirm the date and branch filters are correct.
4. Confirm the approval state is not blocking save or post.
5. Confirm the ledger or item is not missing a dependency.

## Common examples

| Problem | What to check |
| --- | --- |
| Login fails | Email, password, and role assignment |
| Save button is blocked | Required fields, setup data, and validation |
| Report looks wrong | Filters, date range, and opening balances |
| Stock quantity seems off | Item receipt, transfer, and adjustment history |
| Payment cannot be posted | Ledger setup, invoice selection, and approval status |
