## What to learn first

- Company and organization setup.
- User access and role mapping.
- Default preferences.
- Import and export rules.
- Where admin work affects the rest of the guide.
- The Admin overview page shows the other admin areas that do not yet have their own pages.

## Example path

Set the company first.
Create the user role.
Assign the user.
Set the default preferences.
Then review how that role sees the rest of Bizak.
