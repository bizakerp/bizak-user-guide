# Guided Journeys

Use these pages if you want a simple path through Bizak.
Each path stays close to the app pages already documented in this guide.

## How to choose

- New user: start here if you are learning Bizak for the first time.
- Sales user: use this if you create leads, orders, and invoices.
- Procurement user: use this if you buy goods or services.
- Inventory user: use this if you move or adjust stock.
- Finance user: use this if you post journals, reconcile banks, or read financial reports.
- POS user: use this if you run the point of sale screen.
- Admin user: use this if you manage setup, users, and access.
