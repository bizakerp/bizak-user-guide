# Customer Return

This page covers the customer return flow.
Use it when sold goods come back from the customer.

## What the flow does

Sales returns help the team keep the stock and account records aligned.
The return may later link to a credit memo or a debit and credit note.

## What to capture

- Customer name
- Return date
- Source invoice
- Returned quantity
- Reason for return
- Notes

## Example

A customer returns damaged stock after delivery.
The team records the return against the original invoice.
The account team then reviews the matching credit or adjustment entry.
