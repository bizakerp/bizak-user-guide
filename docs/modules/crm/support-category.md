# Support Category

This page covers support category setup.
Use it when support requests need to be grouped.

## What the screen tracks

| Field | Meaning |
| --- | --- |
| `Id` | Category id |
| `Name` | Category name |
| `Parent_Id` | Parent category |

The route supports create, list, update, and view flows.

## Example

A company creates categories for login issues, billing questions, and product help.
Nested categories can sit under a parent category when needed.
