# User Support

This page covers the user support flow.
Use it to log and track support requests.

## What the screen tracks

| Field | Meaning |
| --- | --- |
| `Id` | Support id |
| `Email` | Requester email |
| `Priority` | Priority |
| `CategoryId` | Support category |
| `Subject` | Subject |
| `FileData` | Attachment data |
| `Description` | Request details |
| `OpenDate` | Open date |
| `CloseDate` | Close date |

The screen loads support categories and priority values.
It also uses a rich text editor for the description field.

## Example

A user reports a problem with login.
The support team logs the email, category, priority, and description.
The ticket stays open until the issue is closed.
