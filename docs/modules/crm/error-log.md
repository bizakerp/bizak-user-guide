# Error Log

This page covers the error log view.
Use it when the support team needs to review captured errors.

## What the screen does

- It loads tenant data.
- It supports paging.
- It supports grouping in the grid.
- It can filter by tenant.
- It shows one log detail at a time.

## Example

A support user opens the error log and filters to one tenant.
The list helps them inspect the saved errors before closing the issue.
