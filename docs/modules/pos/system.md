# POS System

This page covers the main POS system screen.
Use it when a session is already open.

## What the route does

- `system` is the main POS screen.
- It is protected by the session guard.
- It assumes the cashier already has an active session.
- It is the primary selling view for counter work.

## Example

A cashier opens the main POS screen after starting a session.
They search items, add them to the sale, and complete payment from the same screen.
