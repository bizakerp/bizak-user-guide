# POS System 2

This page covers the alternate POS system screen.
Use it when the business uses the second POS layout.

## What the route does

- `system-2` is an alternate sales screen.
- It is protected by the session guard.
- It is meant for the same POS flow with a different layout or view.

## Example

Some stores prefer a different screen arrangement.
The alternate system keeps the same POS flow but presents it differently.
