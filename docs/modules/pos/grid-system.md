# Grid POS

This page covers the grid-based POS screen.
Use it when the team wants a grid view for selling.

## What the route does

- `grid-system` is a grid-based POS view.
- It is protected by the session guard.
- It supports the same active-session flow as the main POS screen.

## Example

A retail store uses the grid layout to browse items faster.
The cashier selects items from the grid and continues to payment.
