# Grid POS 2

This page covers the second grid-based POS screen.
Use it when the business uses the alternate grid layout.

## What the route does

- `grid-system-2` is an alternate grid POS view.
- It is protected by the session guard.
- It keeps the same selling flow as the other POS screens.

## Example

A store tests a different grid layout for the same counter work.
The team still follows the same session and payment rules.
