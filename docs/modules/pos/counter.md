# Counter

This page covers POS counter setup.
Use it to create or update a counter record.

## What the screen tracks

| Field | Meaning |
| --- | --- |
| `organisation_id` | Organisation |
| `department_id` | Department |
| `counter_name` | Counter name |
| `description` | Counter description |
| `branch_integration_value` | Branch integration value |

The route supports create, list, and update flows.

## Example

A store creates one counter for each checkout desk.
The counter name helps staff and reports identify the right sales point.
