# Loyalty Point Setup

This page covers loyalty point setup.
Use it when POS sales should earn points.

## What the screen does

- It can apply loyalty by amount or by item.
- It can load item-based rules.
- It stores the point value and the amount or item quantity rule.
- It supports effective from and effective till dates.

## What it tracks

| Field | Meaning |
| --- | --- |
| `loyalty_applicable_on` | Amount or item basis |
| `amount` | Amount threshold |
| `item_id` | Item reference |
| `item_qty` | Item quantity |
| `reward_point` | Reward points |
| `applicable_from` | Start date |
| `applicable_till` | End date |

## Example

The business sets loyalty points for selected items.
Cashiers do not need to calculate points by hand.
