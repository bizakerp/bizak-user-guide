# Modifier Groups and Modifiers

This page covers modifier groups and modifiers.
Use it when items need options or add-ons.

## What the screen tracks

### Modifier group

| Field | Meaning |
| --- | --- |
| `group_name` | Modifier group name |
| `selection_type` | Single or multiple selection |
| `min_selections` | Minimum selections |
| `max_selections` | Maximum selections |
| `is_required` | Required flag |
| `description` | Description |
| `is_active` | Active flag |
| `display_order` | Sort order |

### Modifier

| Field | Meaning |
| --- | --- |
| `modifier_name` | Modifier name |
| `modifier_code` | Modifier code |
| `modifier_group_id` | Modifier group |
| `additional_price` | Extra price |
| `is_default` | Default selection |
| `is_active` | Active flag |
| `display_order` | Sort order |

## Example

A food item can allow cheese or extra sauce.
The group controls the allowed choices.
The modifier stores the actual selectable item.
