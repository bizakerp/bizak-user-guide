# Loyalty Reward

This page covers loyalty reward setup.
Use it when points should be converted to rewards.

## What the screen tracks

| Field | Meaning |
| --- | --- |
| `REWARD_POINT` | Reward points |
| `REWARD_AMOUNT` | Reward amount |
| `INTEGRATION_VALUE` | Integration value |

The page supports add, edit, save, and delete actions.

## Example

The team defines how many points equal one reward amount.
The cashier can then apply the reward policy consistently.
