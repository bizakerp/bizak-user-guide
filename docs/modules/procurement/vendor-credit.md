# Vendor Credit

This page covers the vendor credit flow.
Use it when a vendor issues a credit against a bill or purchase.

## What to capture

| Field | Meaning |
| --- | --- |
| `credit_no` | Credit number |
| `vendor_name` | Vendor name |
| `date` | Credit date |
| `bill_number` | Linked bill |
| `bill_date` | Bill date |
| `ledger_name` | Ledger |
| `location_name` | Location |
| `class_name` | Class |
| `department_name` | Department |
| `project_name` | Project |
| `is_imported` | Imported flag |
| `is_capitalized` | Capitalized flag |
| `details` | Detail lines |
| `account_details` | Account lines |

## Example

A vendor gives a credit for damaged goods.
Bizak records the credit against the related bill.
The purchasing and accounting teams can then reconcile the balance.
