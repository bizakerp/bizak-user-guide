# Salesman Route Map

This page covers the salesman route map.
Use it to map routes to a sales rep.

## What the screen does

- It loads routes for a selected salesperson.
- It saves route assignments for that salesperson.

The service endpoints show that the screen works with route-by-salesman data.

## Example

A manager assigns a set of delivery or visit routes to one sales rep.
The route map makes the assignment visible in the system.
