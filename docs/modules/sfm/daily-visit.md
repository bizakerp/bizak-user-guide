# Daily Visit

This page covers the daily visit flow.
Use it to record field work for the day.

## What the screen tracks

| Field | Meaning |
| --- | --- |
| `market_mapping_id` | Market mapping |
| `is_customer_visit` | Customer visit flag |
| `customer_name` | Customer name |
| `contacted_person` | Contacted person |
| `mobile_number` | Mobile number |
| `check_in_time` | Check-in time |
| `check_out_time` | Check-out time |
| `discussed_agenda` | Agenda discussed |
| `remarks` | Remarks |
| `location` | GPS location |
| `address` | Address |
| `purpose` | Purpose |
| `google_coordinate` | Google coordinate |

The screen uses geolocation.
It also uses geocoding to turn an address into coordinates.

## Example

A rep visits a customer, records the check-in time, and logs the agenda discussed.
The GPS coordinate helps the team verify the visit.
