# Route Plan

This page covers the employee route plan flow.
Use it to assign routes to an employee by date.

## What the screen does

- It loads employees and routes.
- It shows a calendar view.
- It loads planned routes for the selected employee.
- It also loads leaves and holidays.
- It blocks moves to past dates.
- It blocks duplicate routes on the same day.

The route supports `employee-plan` and `employee-plan/:id`.

## Example

A sales manager assigns one route to an employee for each workday.
The calendar helps the team see the full plan for the month.
