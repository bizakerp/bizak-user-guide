# Field Reports

This page covers the field operations report area.
Use it to review field activity reports.

## Main routes in the app

| Route | Purpose |
| --- | --- |
| `eod` | End of day list |
| `eod-report` | End of day report |
| `collection` | Collection list |
| `party-stock` | Party stock view |
| `salesman-activity-report` | Salesman activity report |
| `visit-plan-report` | Visit plan report |
| `attendance-report` | Attendance report |
| `visit-form-report` | Visit form report |

## Example

The manager reviews visit planning, attendance, and end of day information in one area.
That makes field supervision easier.
