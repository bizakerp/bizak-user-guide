# Transfer Fund

This page covers the fund transfer flow.
Use it when money moves between two accounts.

## Before you start

- Confirm both accounts exist.
- Confirm the amount is correct.
- Confirm the user can post bank transactions.
- Confirm any approval rule is clear.

## What the flow does

- It records the source account.
- It records the destination account.
- It keeps the movement visible for reconciliation.
- It supports approval status filters in search.

## Example

The finance team moves cash from a clearing account to the main operating account.
Bizak records the transfer so the account balances stay traceable.
