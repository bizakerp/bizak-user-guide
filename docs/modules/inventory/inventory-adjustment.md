# Inventory Adjustment

This page covers the inventory adjustment flow.
Use it when physical stock and system stock do not match.

## What the screen tracks

| Field | Meaning |
| --- | --- |
| `inventory_adjustment_no` | Adjustment number |
| `adjustment_date` | Adjustment date |
| `account_name` | Adjustment account |
| `location_name` | Location |
| `memo` | Notes |
| `requisition_reference` | Requisition reference |
| `company_name` | Company |
| `project_name` | Project |
| `currency_name` | Currency |
| `exchange_rate` | Exchange rate |
| `department_name` | Department |
| `class_name` | Class |
| `approval_status` | Approval status |
| `adjustment_details` | Adjustment lines |
| `inventory_detail_maps` | Item stock detail mapping |

## Example

A stock count finds missing items.
The team records an adjustment so the ledger and stock report stay correct.
That keeps inventory reports aligned with reality.
