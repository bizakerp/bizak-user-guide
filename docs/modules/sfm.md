# Field Operations

Field operations support route planning, visit tracking, and sales activity outside the office.

## Main routes in the app

| Route | Purpose |
| --- | --- |
| `daily-visit` | Record a daily visit |
| `employee-route-plan` | Plan a route for an employee |
| `customer-in-out` | Record customer in or out activity |
| `customer-visit-summary` | Review visit summaries |
| `customer-visit-detail` | Review visit detail |
| `sales-man-location` | Track sales rep location |
| `party-route-map` | Show party route mapping |
| `salesman-route-map` | Show sales rep route mapping |

## Example

A sales manager can assign a route, track visits during the day, and later review customer visit summaries.
