# Budget

This page covers budget setup.
Use it when the company wants to track planned spend against actuals.

## What the screen tracks

| Field | Meaning |
| --- | --- |
| `ledger_id` | Ledger |
| `fiscal_year_id` | Fiscal year |
| `posting_period_id` | Posting period |
| `class_id` | Class |
| `department_id` | Department |
| `location_id` | Location |
| `project_id` | Project |
| `vendor_id` | Vendor |
| `item_id` | Item |
| `amount` | Budget amount |

## Example

The finance team sets a yearly budget for marketing.
The budget is segmented by period and department.
Reports can then compare the plan to actual spending.
