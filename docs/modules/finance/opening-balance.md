# Opening Balance

This page covers the opening balance flow.
Use it when a new fiscal year or new company setup needs a starting balance.

## What the screen tracks

| Field | Meaning |
| --- | --- |
| `LedgerId` | Ledger id |
| `LedgerName` | Ledger name |
| `FiscalYearId` | Fiscal year |
| `BookClosedId` | Book close reference |
| `DrAmount` | Debit balance |
| `CrAmount` | Credit balance |

## Example

When Bizak goes live, the team loads opening balances for cash, bank, payables, and receivables.
That gives the accounting reports a valid starting point.
